# Smart Lending: Predicting Credit Risk with Machine Learning

## Overview
This project predicts credit risk using machine learning. It demonstrates how to preprocess financial data, engineer features, build predictive models, and evaluate their performance.

The project highlights:
- Data preprocessing and feature engineering on 40K+ financial records  
- Training XGBoost and Neural Network models  
- Optimizing hyperparameters to improve model accuracy and reduce loan defaults  

---

## Key Features

- **Credit Risk Prediction:** Models classify borrowers as high or low risk using XGBoost and Neural Networks.  
- **Hyperparameter Optimization:** Grid search used to improve AUC scores by 15% and reduce loan defaults by 10%.  
- **Data Processing:** Handled missing values, performed one-hot encoding, aggregated features, and normalized data.
